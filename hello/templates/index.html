Sample Inputs

The inputs below to the form will reproduce the example shown in the Demo

Upload file

The file is .csv data file downloaded from Forum and is assumed to follow specific schema. Click here to download a sample .csv file.

LO List

The data file downloaded from Forum does not have the CO column. Therefore, instructor must specify this manually by matching LO to CO. In this field, input the LOs separated by comma, and in the CO list field below, input the COs corresponding to this LO list.

Sample input: mcanalysis, mcmodeling, interpretresults, professionalism, pythonimplementation, caanalysis, camodeling, networkanalysis,networkmodeling

CO list

Sample input: MonteCarlo,MonteCarlo,Simulations,Simulations,Simulations,Cellular Automata,Cellular Automata,Networks,Networks

Assignment titles

The data file also does not have the weights for the assignment and those need to be specified here. List the assignment titles in order in this field and its weights in the following field, all seperated by comma

Sample input: Elevator simulation, Traffic simulation,Network simulation, Final project proposal, Final project

Assignment weights

Sample input: 2, 6, 6, 0, 10

LOs to highlight in LO evolution graph

These are the LOs of interest to highlight (selected) in the evolution graph (see the Evolution tab in the Demo). If no LOs specified, all LOs will be plotted, which may cause clutter.

Sample input: networkanalysis, networkmodeling
